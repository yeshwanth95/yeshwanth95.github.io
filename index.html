Hi, I am Yeshwanth Kumar

a Graduate Scholar in Geoinformatics

What I do

Geospatial Analysis

Being conversant in various remote sensing, photogrammetric & GIS techniques, I can perform extensive analyses such as DEM generation, watershed delineation, LU/LC mapping, digital heritage mapping, structure from motion, etc. and derive meaningful insights from remote sensing & geographic data.

Satellite and LiDAR Data Processing

I am proficient in processing & analysing remote sensing data from a host of optical sensors (Sentinel-2, Landsat-8 OLI, Cartosat, etc.), terrestrial & airborne LiDAR sensors (e.g., FARO Focus S 350) and synthetic aperture radar sensors (Sentinel-1) using Python, R, Javascript, and other software such as ArcGIS, QGIS, CloudCompare, LAStools, ENVI, ERDAS Imagine, SNAP, etc.

Machine Learning

I have a solid understanding of several machine learning & deep learning algorithms (such as KNN, K-means, SVM, MLP, CNN, etc.), using which I can implement end-to-end workflows for classification, segmentation & object detection tasks in satellite imagery and LiDAR point cloud data. I am proficient in using several ML/DL frameworks such as Tensorflow, Pytorch, Keras, scikit-learn, etc.
